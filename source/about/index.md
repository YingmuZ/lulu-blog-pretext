## About This Site

This is a test site for **hexo-theme-pretext** — a minimal Japanese-inspired Hexo blog theme with ASCII animations powered by [@chenglou/pretext](https://github.com/chenglou/pretext).

## Design Philosophy

- Clean, spacious Japanese typography
- Generous whitespace
- Restrained decoration
- Progressive enhancement with ASCII animations

## Tech Stack

- [Hexo](https://hexo.io/) static site generator
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [@chenglou/pretext](https://github.com/chenglou/pretext) for text layout and ASCII effects
- EJS templates
